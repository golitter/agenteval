# 目标智能体分析报告
## 1. 智能体的任务
该智能体是麦当劳的智能助手，主要设计目的是为用户提供便捷的麦当劳优惠券和活动管理服务。其主要任务包括：

1. 优惠券管理：帮助用户查询、领取和使用麦当劳的各种优惠券
2. 活动信息查询：提供麦当劳中国当月的营销活动日历信息
3. 时间信息获取：提供准确的时间信息，方便用户安排参与活动

具体功能包括：
- 查询可领取优惠券：查看当前有哪些优惠券可以领取
- 自动领券：一键领取所有可用的优惠券
- 管理我的优惠券：查看用户已经拥有的优惠券，包括有效期和使用条件
- 查看活动日历：了解麦当劳当前和未来的营销活动
- 获取时间信息：确保用户不错过任何优惠和活动时间

该智能体的设计目标是让用户更方便地享受麦当劳的各种优惠和活动，节省用户的时间和精力，最大化用户的麦当劳消费体验。

## 2. 智能体使用的工具列表及其功能描述
该智能体使用了以下5个工具来完成其任务：

## 1. campaign-calender - 营销活动日历查询工具
**功能**：查询麦当劳中国当月的营销活动日历
- 返回进行中、往期和未来日期的活动
- 适用于查看用户进行中和即将到来的可参与活动
- 可查询用户对活动的订阅状态
- 支持指定查询基准日期（格式：yyyy-MM-dd）
  - 若传入日期，则以该日为锚点，返回当天及时间线上与其最邻近的前后各一个有活动的日子（共三天）
  - 若不填，默认返回当前月所有活动

## 2. available-coupons - 可领取优惠券查询工具
**功能**：查询用户当前可领取的麦麦省优惠券列表
- 返回券名称、图片、状态和促销标签
- 当用户询问有什么优惠、可以领什么券时使用
- 帮助用户发现当前可用的优惠机会

## 3. auto-bind-coupons - 自动领券工具
**功能**：自动领取麦麦省所有当前可用的麦当劳优惠券
- 无需指定具体的优惠券和couponId
- 系统会自动领取用户可领的所有券
- 使用场景：
  - 用户说"帮我领券"
  - "自动领取优惠券"
  - "一键领券"

## 4. my-coupons - 我的优惠券查询工具
**功能**：查询用户拥有的可用优惠券
- 就像打开麦当劳APP的"我的优惠券"页面
- 能看到所有可以用来点餐的优惠券列表
- 主要使用场景：
  - 用户想知道自己有哪些优惠券可以用
  - 检查优惠券的有效期和使用条件
  - 查看优惠券数量和状态
- 支持分页查询（默认第1页，最多5页，每页最多200条）

## 5. now-time-info - 当前时间信息工具
**功能**：获取当前服务器时间信息
- 返回当前服务器的完整时间信息，包括：
  - 时间戳（毫秒级）
  - 格式化的日期时间
  - 年月日信息
  - 时区和UTC时间
- 使用场景：
  - 当需要知道当前时间时
  - 用户需要指定日期查询活动日历时

## 智能体工作流程和策略

### 智能工具选择机制
智能体会根据用户的问题类型自动选择合适的工具：
- 询问优惠券 → 使用 `available-coupons` 或 `my-coupons` 工具
- 想要领券 → 使用 `auto-bind-coupons` 工具
- 查询活动 → 使用 `campaign-calender` 工具
- 需要时间信息 → 使用 `now-time-info` 工具

### 问题理解与分类
智能体会先分析用户意图，然后：
- 如果是简单查询（如"有什么优惠"），直接调用相应工具
- 如果是复杂需求，可能需要组合多个工具或提供分步指导
- 对于模糊的问题，智能体会先澄清再采取行动

## 使用场景和限制

### 主要使用场景：
- 当用户想领取麦当劳优惠券时
- 当用户想查看自己有哪些优惠券可用时
- 当用户想了解麦当劳当前有什么活动时
- 当用户需要确认当前时间以查询相关活动时

### 使用限制：
1. 这些功能主要针对麦当劳中国的用户
2. 优惠券和活动的可用性可能因地区、时间等因素有所不同
3. 部分功能可能需要用户在麦当劳APP或相关平台上有相应的账户

### 目前没有的功能：
- 直接下单点餐
- 查询具体门店信息
- 处理支付相关操作
- 用户账户管理功能

## 用户数据处理说明

### 数据访问范围：
智能体仅能访问以下类型的用户数据：
- 优惠券信息（通过 `my-coupons` 工具）
- 活动订阅状态（通过 `campaign-calender` 工具）
- 优惠券领取状态（通过 `available-coupons` 工具）

### 数据处理原则：
- **最小必要原则**：只访问完成任务所需的最少数据
- **用户授权**：所有数据访问都基于用户明确请求或同意
- **目的限定**：数据仅用于提供麦当劳相关服务

### 数据安全：
- 通过麦当劳官方API进行数据交互
- 不存储用户敏感信息
- 遵循麦当劳的数据保护政策

### 透明度：
- 当需要访问用户数据时，智能体会明确告知
- 用户可以随时停止使用相关功能
- 所有数据处理都符合麦当劳的隐私政策

- 确保响应及时准确
